# Astral Purple

A premium purple glass theme for BetterDiscord with custom background and smooth interactive elements.

## ✨ Features

- **Custom Background:** Stunning purple background with dynamic overlays.
- **Glassmorphism:** Transparent Discord UI with frosted glass sidebars, panels, and menus.
- **Interactive Elements:** Smooth hover effects on servers, chat messages, and the member list.
- **Custom Scrollbar:** Sleek, thin scrollbar matching the theme's purple color palette.
- **Clean Interface:** Hidden typing indicator and attachment button circles for a clutter-free look.
- **Styled Inputs & Buttons:** Custom styling with box-shadows and gradients for inputs, search bars, and primary buttons.

## 📥 Installation

1. Ensure you have [BetterDiscord](https://betterdiscord.app/) installed.
2. Download the `AstralPurple.theme.css` file.
3. Open your BetterDiscord Themes folder:
   - Go to Discord Settings > Themes > Open Themes Folder.
4. Place the `AstralPurple.theme.css` file into the folder.
5. Enable the "Astral Purple" theme in Discord settings.

## 🎨 Customization

You can change the core colors by modifying the CSS variables located at the top of the file in the `:root` block:

```css
:root {
  --ap-p1: #8A5CFF;
  --ap-p2: #6E3BFF;
  --ap-p3: #B89CFF;
  --ap-p4: #3d1fa8;
}
```

To change the background image, update the `background-image` URL under the `body` selector.
